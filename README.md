# Iris Data Visualization

## Problem Statement
Visualize patterns and relationships in the Iris dataset using different types of charts and graphical analysis.

## Dataset
Iris Dataset from Kaggle

## Tasks Performed
- Explored dataset structure and summary statistics
- Created bar chart for species distribution
- Created histogram for feature distribution
- Created scatter plot to analyze feature relationships
- Compared features using pair plots
- Visualized species-wise distribution using box plots
- Generated correlation heatmap

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

## Results
Successfully visualized the distribution and relationships of Iris flower features. Identified clear differences among species, especially through petal measurements and correlation analysis.

## Conclusion
Data visualization helped uncover patterns in the dataset and provided a better understanding of feature relationships and species classification.
